# graphcast_project
Les modèles météorologiques conventionnels (tels que ECMWF, GFS), bien que robustes, atteignent aujourd'hui leurs limites face aux exigences des marchés énergétiques modernes : mises à jour trop espacées, résolutions spatiales parfois insuffisantes, et manque de spécificité sur certaines variables critiques pour les EnR. Nous allons resoudre çà !
